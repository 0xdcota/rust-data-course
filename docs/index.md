# Data Engineering with Rust

Welcome to the Data Engineering with Rust course sandbox. This repository contains practical exercises and projects for learning data engineering concepts using Rust.

## Overview

This course is organized into modules, with each module corresponding to a week of learning. Each module contains:
- **Source code** with practical examples
- **Build configuration** via Makefiles
- **Exercises** to reinforce concepts

## Course Modules

- [Module 1](./modules.md#module-1) - Fundamentals
- [Module 2](./modules.md#module-2) - Core Concepts
- [Module 3](./modules.md#module-3) - Advanced Topics
- [Module 4](./modules.md#module-4) - Practical Projects

## Getting Started

To get started with this course:

1. Clone the repository:
   ```bash
   git clone https://github.com/username/rust-data-course
   cd rust-data-course
   ```

2. Navigate to a module:
   ```bash
   cd content/module-1
   ```

3. Review the exercises and complete them using Rust
