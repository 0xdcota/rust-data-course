## General

This repository is a sandbox for the `data-engineering-rust` course.    
   
The folders are organized by module, and each module corresponds to a week in the course. 

```shell
.
├── content
│   ├── module-1
│   ├── module-2
│   ├── module-3
│   └── module-4
└── README.md
```
